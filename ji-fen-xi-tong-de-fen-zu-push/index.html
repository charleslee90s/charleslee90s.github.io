##逻辑

  1. 优先发送分值最高的内容； 点击PUSH +1分，未点击-6分； 最高分100分，最低分0分；
  2. 出现分值为0时，默认弹内容；
  3. 后续新增通知类型时，默认分值需要根据不同用户分组，进行设置；默认为81分
  4. Seven的每日文案工作量：内容1条，积分刺激4条；
  5. MV、音频、短视频，跳转至指定歌曲； 积分刺激，跳转至指定页面；
st=>start: 开始
in=>inputoutput: 输入数据 a，b，c
cond1=>condition: a < b
op1=>operation: 交换a，b的值
cond2=>condition: b < c
op2=>operation: 交换b,c的值
out=>inputoutput: 输出数据 a，b，c
e=>end: 结束
st->in->cond1
cond1(yes, right)->op1->cond2
cond1(no)->cond2
cond2(yes, right)->op2->out->e
cond2(no)->out->e
