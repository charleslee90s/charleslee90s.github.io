Tag 相关接口

现有的艺术家，专辑，歌曲中包含此 tag 的内容

  • Tag_Info

  • Tag_TopArtists
    现有的艺术家中包含此 Tag 的内容，并提供给 Artists_Similar 使用

  • Tag_TopAlbums
    现有的专辑中包含此 Tag 的内容，并提供给 Albums_Similar 使用

  • Tag_TopTracks
    现有的歌曲中包含此 Tag 的内容，并提供给 Track_Similar 使用
