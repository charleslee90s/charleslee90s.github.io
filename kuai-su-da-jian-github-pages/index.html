快速搭建 GitHub Pages

2019-11-01 / 1 min read
  • 最初如果想搭建简单的 个人Blog，用 GitHub Pages 就是最Easy and Free 的方式了。研究这个玩意的最初目的其实想作为我个人工作用的需求沟通工具用来给开发使用，但是需要给对应的项目单独建立 Pages以满足权限管理的需要。

一，创建 GitHub 账号

通过邮箱进行注册

通过转动图片保证动物是正向的

这种中立的情况则有对有错:D

二，创建 Repository

在 Repository name 项需要填写的格式为 account.github.io

三，主题

建好 Repository 后，进入 Settings 进行配置

可以选择一个 GitHub 预置的主题进行配置

四，编辑内容

选择后就跳转到了内容编辑器内，可用 Html or Markdown 进行编写

完成编辑保存后就可以访问了，url格式为 account.github.io
